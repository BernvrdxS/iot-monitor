IoT Monitor

Ambiente completo de monitoramento IoT desenvolvido para a disciplina de Cloud Computing, integrando tecnologias modernas de containerização e automação.

Detalhes do Projeto

Visão Geral

O IoT Monitor é um ambiente desenvolvido para demonstrar conceitos de Docker, Linux, Apache e Shell Script aplicados ao monitoramento de dispositivos IoT. O projeto integra todas as camadas de uma solução real: infraestrutura em container, servidor web, coleta de dados e administração automatizada via scripts.

  • Coleta e armazenamento de dados IoT
  • Painel de visualização web com Apache
