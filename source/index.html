Sobre o Projeto

Desenvolvido para a disciplina de Cloud Computing, o IoT Monitor integra Docker, Linux, Apache e Shell Script para gerenciamento e monitoramento de um ambiente IoT completo.

O sistema realiza coleta contínua de dados dos sensores, exibe métricas em tempo real e oferece uma camada de administração automatizada via scripts de shell.
